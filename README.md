# GeoIndia Nature Organization Website

Welcome to the Git repository for GeoIndia, a nature organization dedicated to promoting environmental awareness and conservation efforts in India. This repository contains the source code for our website, which is built using HTML, CSS, JavaScript, and jQuery. Our website features responsive and modern designs to provide an engaging user experience across various devices.

## Features

- Responsive design: Ensures optimal viewing experience across different devices and screen sizes.
- Modern layouts: Utilizes contemporary design principles to create visually appealing pages.
- Integration with Razorpay: Allows seamless online payment processing for donations and memberships.
- Interactive animations: Incorporates jQuery for dynamic and engaging animations throughout the website.

## Contents

The repository includes the following files and directories:

- `index.html`: The homepage of the website.
- `about.html`: Information about GeoIndia and its mission.
- `projects.html`: Details about ongoing and past projects.
- `gallery.html`: A collection of photos showcasing India's diverse ecosystems.
- `contact.html`: Contact information and a form for inquiries.
- `css/`: Directory containing stylesheets for styling the website.
- `js/`: Directory containing JavaScript files, including jQuery for animations.
- `img/`: Directory containing images used in the website.
- `fonts/`: Directory containing custom fonts used in the website.
- `vendor/`: Directory containing third-party libraries, including Razorpay integration scripts.

## Getting Started

To run the website locally or deploy it to a web server, simply clone this repository to your local machine using the following command:

```
git clone https://github.com/rishwal/geoindia.git
```

Once cloned, you can open the `index.html` file in a web browser to view the website locally. Ensure that you have an internet connection if you intend to test the Razorpay integration.
